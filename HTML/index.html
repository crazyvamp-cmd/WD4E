My current favorite FOSS software

Since I've already mentioned liking FOSS, here are some of the pieces of it that I especially like, grouped into separate lists...

Operating Systems

  • Linux Mint (yes, I know...)
  • Kubuntu (its Dolphin is a lifesaver!)

Writing software

  • LibreOffice (general writing, especially with the WritingTool extension - if your language is supported)
  • FocusWriter (if you want a really basic writing piece of software which will basically make you write and do nothing else)
  • Trelby (amazing for screenwriting)

Graphic Design

  • GIMP (seriously! especially with certain plug-ins - and do not forget to read its documentation!)
  • Inkscape (amazing for vector work)
  • Scribus (not perfect, but decent for desktop publishing - its Lorem Ipsum and palette generation tools are especially awesome)
  • WebStudio (just try it, I beg you!)

General coding

  • Geany (lightweight as heck, and has a plethora of useful plug-ins!)
  • Kate (basically, a Geany equivalent for KDE Plasma!)
  • Git (once you learn how to use it, it rocks!)
